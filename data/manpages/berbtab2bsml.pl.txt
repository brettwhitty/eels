NAME
       berbtab2bsml.pl  - convert info stored in BER btab files into BSML documents

SYNOPSIS
       USAGE:  berbtab2bsml.pl -b btab_dir -o blastp.bsml -d bsml_dir

OPTIONS
       •   --bsml_dir,-d [REQUIRED]  Dir containing BSML documents (repository)

       •   --output,-o [REQUIRED] output BSML file containing bit_score information

       •   --btab_dir,-b [REQUIRED] Dir containing btab files

       •   --max_hsp_count [REQUIRED] Maximum number of HSPs stored per alignment

       •   --gzip_output [OPTIONAL] A non-zero value will result in compressed bsml output.  A .gz
               extension will be added to the output file name if it doesn't already exist.

       •   --query_file_path,-q [OPTIONAL] The input fasta file to ber.

       •   --class,-c The ref/comp sequence type.  Default is 'assembly'

       •   --project_abbreviation,-a The project abbreviation, used in id generation. See Ergatis::IdGenerator for details.

       •   --help,-h This help message

DESCRIPTION
       berbtab2bsml.pl is designed to convert information in ber btab files into BSML documents.

       NOTE:

       Calling the script name with NO flags/options or --help will display the syntax requirement.
