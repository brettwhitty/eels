NAME
       prodigal2bsml.pl - Creates a bsml document from prodigal raw output

SYNOPSIS
       USAGE: prodigal2bsml.pl
                   --input_list=/path/to/some/prodigal.raw.list
                   --input_file=/path/to/some/prodigal.raw
                   --output=/path/to/transterm.bsml
                   --id_repository=/path/to/id_repository
                   --fasta_input=/path/to/prodigal/input.fsa
                 [ --log=/path/to/file.log
                   --debug=4
                   --help
                 ]

OPTIONS
       --input_list,-i
           Input list of prodigal raw output files (.predict)

       --input_file,-f
           Input prodigal raw file (.predict)

       --output,-o
           The output bsml file.

       --id_repository,-r
           Id repository for use by Workflow::IdGenerator.pm

       --fasta_input,-a
           The input file that was used as input for the prodigal run

       --log,-l
           Logfile.

       --help,-h
           Print this message

DESCRIPTION
       This script is used to convert the output from a prodigal search into BSML.

INPUT
       You define the input file using the --input option.  This file does not need any special file extension.  The regular output of prodigal looks like this:

       # Sequence Data: seqnum=1;seqlen=5127136;seqhdr="gec3273.pseudomolecule.11" # Model Data: version=Prodigal.v2.50;run_type=Single;model="Ab
       initio";gc_cont=50.70;transl_table=4;uses_sd=1
        >17_20759_21505_+
        >18_21508_22068_-
        >19_22103_22444_-
        >20_22579_22905_+

       Where the columns from left to right separated by '_' contain: prediction_id   start_position   end_position   strand start_position is always less than the end_position

OUTPUT
       After parsing the input file, a file specified by the --output option is created.  This script will fail if it already exists.  The file is created, and temporary IDs are
       created for each result element.  They are only unique to the document, and will need to be replaced before any database insertion.

       Base positions from the input file are renumbered so that positions start at zero.

       NOTE: This script is created by modifying Kevin's glimmer32bsml.pl for prodigal
