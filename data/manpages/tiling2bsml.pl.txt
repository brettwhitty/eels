NAME
       tiling2bsml.pl  - convert show_tiling output into BSML sequence mapping documents

SYNOPSIS
       USAGE:  tiling2bsml.pl -t tilings_file -b path_to_bsml_repository -o output_file_bsml

OPTIONS
       •   --tilingPath,-t [REQUIRED] file containing tiling path data from show_tiling

       •   --bsml_repository,-b [REQUIRED] filepath to the bsml repository

           --gzip_output,-g [OPTIONAL] a non-zero value will result in compressed bsml output

       •   --output,-o [REQUIRED] output BSML file containing coverage information

       •   --help,-h This help message

DESCRIPTION
       tiling2bsml.pl is designed to convert tiling path data from show_tiling into BSML sequence mapping documents.

       The tiling path input should be tab delimited with the following fields. The first line encode the reference assembly id, starting with a '>'

       # 0 - start in ref # 1 - end in ref # 2 - distance to next contig # 3 - length of this contig # 4 - alignment coverage # 5 - identity # 6 - orientation # 7 - id
