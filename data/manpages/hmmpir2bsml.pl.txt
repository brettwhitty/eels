NAME
       hmmpir2bsml.pl - convert hmmpir raw output to BSML

SYNOPSIS
       USAGE: hmmpir2bsml.pl
               --input=/path/to/somefile.hmmpir.raw
               --output=/path/to/somefile.hmmpir.bsml
             [ --fasta_input=/path/to/hmmpir/input.fsa
               --gzip_output=1
               --log=/path/to/some.log
               --debug=4
               --help
             ]

OPTIONS
       --input,-i
           Input raw alignment file from an hmmpir search.

       --output,-o
           Output BSML file

       --fasta_input,-f
           Optional.  The input file used in the hmmpir run that created the raw file.

       --gzip_output,-g
           Optional.  Compress the bsml output if a non-zero value.

       --debug,-d
           Debug level.  Use a large number to turn on verbose debugging.

       --log,-l
           Log file

       --help,-h
           This help message

DESCRIPTION
       This script is used to convert the raw alignment output from an hmmpir search into BSML.

INPUT
       The input file passed to this script must be a raw alignment file generated by hmmpir.  Define the input file using the --input option.

       Illegal characters will be removed from the IDs for the query sequence and subject hit if necessary to create legal XML id names.  For each element, the original, unmodified
       name will be stored in the "title" attribute of the Sequence element.  You should make sure that your ids don't begin with a number.  This script will successfully create a BSML
       document regardless of your ID names, but the resulting document may not pass DTD validation.

OUTPUT
       The BSML file to be created is defined using the --output option.  If the file already exists it will be overwritten.
